# DeFi Dashboard ⚡

A live cryptocurrency dashboard built with vanilla JavaScript and the CoinGecko API.

## Features
- Real-time prices for 8 major cryptocurrencies
- 24h price change tracking
- Sample portfolio value calculator
- Live trending coins feed
- Auto-refreshes every 60 seconds

## Tech Stack
HTML, CSS, JavaScript, CoinGecko API
